# Release Checklist

## Pre-Release
- [ ] Version number updated
- [ ] Changelog prepared
- [ ] QA sign-off
- [ ] Security review completed

## App Store / Play Store
- [ ] Screenshots updated
- [ ] App metadata reviewed
- [ ] Privacy policy link verified
- [ ] Support URL verified

## Post-Release
- [ ] Monitor crash reports
- [ ] Verify key user flows
- [ ] Announce release
